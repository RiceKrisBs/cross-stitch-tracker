# Cross-Stitch Tracker

A personal web application for tracking cross-stitch projects, floss inventory, work sessions, and shopping lists.

## Tech Stack

- **Backend:** FastAPI (Python)
- **Database:** SQLite with SQLModel ORM
- **Frontend:** Jinja2 templates + htmx + Pico CSS
- **Testing:** pytest

## Features

- Pattern management with floss color requirements
- Project tracking (planned, in progress, completed)
- Work session tracking with timer and analytics
- Floss inventory management with storage locations
- Shopping list for floss to purchase
- User authentication
